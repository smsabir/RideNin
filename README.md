#RideNin is a ticket booking app made with react.

Technology used:
React, Bootstrap, React hooks, React Router, FireBase.

#Map used from:
Mapbox.


Features of this project:

* Auth using firebase.
* React Router
* React Private Routing 
* React form
* Firebase deploy
* Context API
* Conditional Rendering


Hosted on firebase:
https://ridenin.web.app/
